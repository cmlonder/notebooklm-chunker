# Story 7: System Health & Diagnostics (The Doctor)

## Business Need
Ensure the environment is ready. Think of this like the "Settings > Battery" or "About This Mac" page.

## UI Elements & Interactions
- **Health Grid:** 4 simple icons with "Pass/Fail" text.
- **"Setup" Button:** A simple, text-based button like "Update" or "Install" next to any red item.
- **Detail View:** Clicking a health item reveals a small "System Report" style text area.

## Designer Notes (Apple Minimalism)
- **System Fonts:** Use the native system font stack.
- **Status Colors:** Use Apple's standard Green and Red (not too neon).
