# NotebookLM Chunker Desktop

Electron desktop client for `notebooklm-chunker`.

For full documentation, features, and installation instructions, see the main [README](../README.md).

## Development

```bash
cd desktop
npm install
npm run dev
```

## Tests

```bash
cd desktop
npm test
```

## Build

```bash
npm run build:mac    # macOS
npm run build:win    # Windows
npm run build:linux  # Linux
```
