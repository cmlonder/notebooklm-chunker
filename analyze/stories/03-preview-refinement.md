# Story 3: The Refinement Editor (Preview)

## Business Need
Verify and tweak the chunks. It should feel like macOS "Quick Look" meets a modern Markdown editor.

## UI Elements & Interactions
- **Sidebar (The Catalog):** A clean vertical list of chunks. Selected chunk has a subtle "active" background.
- **Content Area:** A distraction-free text area with high-contrast typography.
- **"Split Here" Action:** When hovering between lines in the editor, a thin horizontal line with a `plus` icon appears to allow manual splitting.
- **Top Toolbar:** Minimalist icons for [Save to Folder], [Share], and [Delete].

## Designer Notes (Apple Minimalism)
- **Split View:** Use a 1-pixel separator between the sidebar and editor.
- **Padding:** Give the text in the editor plenty of room to "breathe" (generous margins).
- **Icons:** Use SF Symbols for all actions (e.g., `square.and.arrow.up` for Export).
