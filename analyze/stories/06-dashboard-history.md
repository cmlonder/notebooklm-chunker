# Story 6: Workflow Persistence (History & Resume)

## Business Need
Access past work. This should look like a clean "Recent Documents" view in Pages or Keynote.

## UI Elements & Interactions
- **Recent List:** A table with plenty of row height.
- **"Resume" Highlight:** If a run is unfinished, it should have a subtle "un-filled" progress bar to encourage completion.
- **Search Bar:** A standard, rounded search field at the top right.

## Designer Notes (Apple Minimalism)
- **Empty State:** If no history, show a "Start your first project" call to action.
- **Alternating Rows:** Avoid harsh colors; use very subtle gray for alternate rows or no borders at all.
